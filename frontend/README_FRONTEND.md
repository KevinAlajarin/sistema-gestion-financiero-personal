# Personal Finance Manager - Frontend

Frontend desarrollado con React + Vite + Tailwind CSS.

## Estructura
- `/src/components`: Componentes reutilizables (UI Kit y Lógica).
- `/src/pages`: Vistas principales enrutadas.
- `/src/services`: Capa de conexión con la API (Axios).
- `/src/context`: Estado global (Perfil activo).

## Scripts
- `npm run dev`: Iniciar servidor de desarrollo.
- `npm run build`: Construir para producción.
